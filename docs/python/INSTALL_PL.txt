MAXScript GUI Editor v1.18 — Instrukcja instalacji (Windows)
=============================================================

WYMAGANIA
---------
  • Windows 10 / 11
  • Python 3.9 lub nowszy
      https://www.python.org/downloads/
      → Podczas instalacji zaznaczyć "Add Python to PATH"
  • 3ds Max 2022 lub nowszy  (tylko dla TCP bridge, opcjonalnie)


CZĘŚĆ 1 — INSTALACJA APLIKACJI PYTHON
--------------------------------------
  1. Rozpakować plik  maxscript-gui-editor-python-1.18.zip.

  2. Dwukrotnie kliknąć  install.bat  w rozpakowanym folderze.

  3. Skrypt automatycznie:
       • Sprawdzi wersję Pythona (wymagana 3.9+)
       • Utworzy wirtualne środowisko:
           %LOCALAPPDATA%\MAXScriptGUIEditor\
       • Zainstaluje PySide6
       • Utworzy skrót na pulpicie
       • Doda wpis do menu Start

  4. Uruchomić aplikację przez skrót "MAXScript GUI Editor" na pulpicie.


CZĘŚĆ 2 — INSTALACJA NASŁUCHIWACZA BRIDGE W 3DS MAX  (opcjonalnie)
-------------------------------------------------------------------
Umożliwia bezpośrednie wysyłanie wygenerowanego kodu do 3ds Max.

  1. Skopiować plik  max_bridge_listener.ms  z rozpakowanego folderu
     do stałej lokalizacji, np.:
       C:\MaxScripts\max_bridge_listener.ms

  2. Uruchomić raz w 3ds Max:
       Scripting > Run Script > wybrać max_bridge_listener.ms

     W oknie MAXScript Listener pojawi się:
       === MAXScript GUI Editor Bridge Listener v1.18 started on port 27120 ===

  3. Opcjonalnie — automatyczny start z 3ds Max:
       Skopiować plik do:
       %LOCALAPPDATA%\Autodesk\3dsMax\<Wersja>\ENU\scripts\startup\
       Nasłuchiwacz będzie wtedy uruchamiany automatycznie z 3ds Max.

  4. Przetestować połączenie w edytorze GUI:
       Code > Bridge Settings…  — domyślnie: 127.0.0.1 : 27120
       Kliknąć "⬤ Ping" — zmieni kolor na zielony przy powodzeniu.

  Ręczne zatrzymanie nasłuchiwacza (MAXScript Listener w 3ds Max):
       stopBridgeListener()


DEINSTALACJA
------------
  • Usunąć wirtualne środowisko:  %LOCALAPPDATA%\MAXScriptGUIEditor\
  • Ręcznie usunąć skrót z pulpitu
  • Wpis w menu Start:  prawym przyciskiem > Odinstaluj


STANDALONE EXE  (opcjonalnie, bez wymagania Pythona u użytkownika)
------------------------------------------------------------------
  pip install pyinstaller
  pyinstaller maxscript_gui_editor.spec
  → Wynik:  dist\MAXScriptGUIEditor\MAXScriptGUIEditor.exe
