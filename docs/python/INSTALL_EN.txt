MAXScript GUI Editor v1.18 — Installation Guide (Windows)
==========================================================

REQUIREMENTS
------------
  • Windows 10 / 11
  • Python 3.9 or newer
      https://www.python.org/downloads/
      → Check "Add Python to PATH" during installation
  • 3ds Max 2022 or newer  (only for TCP bridge, optional)


PART 1 — INSTALL THE PYTHON APP
--------------------------------
  1. Extract  maxscript-gui-editor-python-1.18.zip.

  2. Double-click  install.bat  in the extracted folder.

  3. The script automatically:
       • Checks your Python version (3.9+ required)
       • Creates a virtual environment at:
           %LOCALAPPDATA%\MAXScriptGUIEditor\
       • Installs PySide6
       • Creates a Desktop shortcut
       • Creates a Start Menu entry

  4. Launch the app via the "MAXScript GUI Editor" Desktop shortcut.


PART 2 — INSTALL THE 3DS MAX BRIDGE LISTENER  (optional)
---------------------------------------------------------
Allows sending generated code directly to 3ds Max.

  1. Copy  max_bridge_listener.ms  from the extracted ZIP folder
     to a permanent location, e.g.:
       C:\MaxScripts\max_bridge_listener.ms

  2. Run it once inside 3ds Max:
       Scripting > Run Script > select max_bridge_listener.ms

     The MAXScript Listener window will show:
       === MAXScript GUI Editor Bridge Listener v1.18 started on port 27120 ===

  3. Optional — auto-start with 3ds Max:
       Copy the file to:
       %LOCALAPPDATA%\Autodesk\3dsMax\<Version>\ENU\scripts\startup\
       The listener will then start automatically with 3ds Max.

  4. Test the connection in the GUI Editor:
       Code > Bridge Settings…  — default: 127.0.0.1 : 27120
       Click "⬤ Ping" — turns green on success.

  To stop the listener manually (MAXScript Listener in 3ds Max):
       stopBridgeListener()


UNINSTALLATION
--------------
  • Delete the virtual environment:  %LOCALAPPDATA%\MAXScriptGUIEditor\
  • Delete the Desktop shortcut manually
  • Start Menu entry:  right-click > Uninstall


STANDALONE EXE  (optional, no Python required on the end-user machine)
----------------------------------------------------------------------
  pip install pyinstaller
  pyinstaller maxscript_gui_editor.spec
  → Output:  dist\MAXScriptGUIEditor\MAXScriptGUIEditor.exe
