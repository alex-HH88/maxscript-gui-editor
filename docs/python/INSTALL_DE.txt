MAXScript GUI Editor v1.18 — Installationsanleitung (Windows)
==============================================================

VORAUSSETZUNGEN
---------------
  • Windows 10 / 11
  • Python 3.9 oder neuer
      https://www.python.org/downloads/
      → Installer-Option "Add Python to PATH" aktivieren
  • 3ds Max 2022 oder neuer  (nur für TCP-Bridge, optional)


TEIL 1 — PYTHON APP INSTALLIEREN
---------------------------------
  1. ZIP-Datei  maxscript-gui-editor-python-1.18.zip  entpacken.

  2. Im entpackten Ordner  install.bat  doppelklicken.

  3. Das Skript führt automatisch aus:
       • Python-Version prüfen (mind. 3.9)
       • Virtuelle Umgebung anlegen:
           %LOCALAPPDATA%\MAXScriptGUIEditor\
       • PySide6 installieren
       • Desktop-Shortcut erstellen
       • Startmenü-Eintrag anlegen

  4. App über den Desktop-Shortcut "MAXScript GUI Editor" starten.


TEIL 2 — 3DS MAX BRIDGE-LISTENER INSTALLIEREN  (optional)
----------------------------------------------------------
Ermöglicht das direkte Senden von generiertem Code an 3ds Max.

  1. Datei  max_bridge_listener.ms  aus dem entpackten ZIP-Ordner
     an einen festen Speicherort kopieren, z. B.:
       C:\MaxScripts\max_bridge_listener.ms

  2. In 3ds Max einmalig ausführen:
       Scripting > Run Script > max_bridge_listener.ms auswählen

     Der MAXScript Listener zeigt nach dem Start:
       === MAXScript GUI Editor Bridge Listener v1.18 started on port 27120 ===

  3. Optional — automatischen Start einrichten:
       Datei nach  %LOCALAPPDATA%\Autodesk\3dsMax\<Version>\ENU\scripts\startup\
       kopieren. Dann startet der Listener automatisch mit 3ds Max.

  4. Verbindung im GUI Editor testen:
       Code > Bridge Settings…  — Standard: 127.0.0.1 : 27120
       Schaltfläche "⬤ Ping" klicken — wird grün bei Erfolg.

  Listener manuell beenden (MAXScript Listener in 3ds Max):
       stopBridgeListener()


DEINSTALLATION
--------------
  • Virtuelle Umgebung löschen:  %LOCALAPPDATA%\MAXScriptGUIEditor\
  • Desktop-Shortcut manuell löschen
  • Startmenü-Eintrag:  Rechtsklick > Deinstallieren


STANDALONE EXE  (optional, kein Python beim Endnutzer nötig)
------------------------------------------------------------
  pip install pyinstaller
  pyinstaller maxscript_gui_editor.spec
  → Ausgabe:  dist\MAXScriptGUIEditor\MAXScriptGUIEditor.exe
