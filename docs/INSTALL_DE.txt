MAXScript GUI Editor
====================
Visueller Rollout- / MacroScript-Designer für Autodesk 3ds Max
--------------------------------------------------------------
Zwei Editionen:
  A) MAXScript Edition  — läuft direkt in 3ds Max
  B) Python Standalone  — Desktop-App mit WYSIWYG-Canvas & TCP-Bridge

==============================================================
  A) MAXSCRIPT EDITION
==============================================================

VORAUSSETZUNGEN
  - Autodesk 3ds Max 2022 oder neuer

INSTALLATION
  1. ZIP  maxscript-gui-editor-<version>.zip  entpacken.
  2. 3ds Max öffnen.
  3. Installer ausführen — eine der folgenden Methoden:
       • Datei  installer-<version>.ms  auf den Viewport ziehen
       • Scripting > Run Script > installer-<version>.ms wählen
  4. Bestätigungsdialog lesen und schließen.
     Das Tool ist jetzt als MacroScript registriert (Kategorie "Tools").

TOOLBAR-SCHALTFLÄCHE EINRICHTEN
  - Customize > Customize User Interface > Tab "Toolbars"
  - Kategorie: Tools
  - "MAXScript GUI Editor" per Drag & Drop auf eine Toolbar ziehen.
  Tipp: Im Tab "Keyboard" kann auch ein Tastaturkürzel vergeben werden.

BEDIENUNG
  Links  – "Add Control"-Buttons: Control-Typ wählen und hinzufügen.
  Rechts – Eigenschaften des gewählten Controls bearbeiten.
  Unten  – Rollout-Einstellungen, Undo/Redo, Speichern, Code generieren.

  Control hinzufügen    → Button im linken Panel klicken
  Control auswählen     → in der Liste links anklicken
  Eigenschaften ändern  → rechtes Panel (typ-spezifisch)
  Event-Handler         → Tab "Event Handlers" im linken Panel
  Code generieren       → "Generate Code" → Code in Zwischenablage
  MacroScript-Modus     → Tab "MacroScript Config" aktivieren

SPEICHERN / LADEN
  "Save Layout"   → speichert Design als .dat-Datei
  "Load Layout"   → lädt gespeichertes Design
  "Import Code"   → liest bestehenden MAXScript-Rollout-Code ein
                    (Basis-Parser; komplexe Ausdrücke ggf. manuell korrigieren)

DEINSTALLATION
  1. Ordner löschen:
       %LOCALAPPDATA%\Autodesk\3dsMax\<version>\ENU\scripts\MAXScriptGUIEditor\
  2. Toolbar-Button entfernen:
       Customize > Customize User Interface > Button per Rechtsklick entfernen.

==============================================================
  B) PYTHON STANDALONE EDITION
==============================================================

VORAUSSETZUNGEN
  - Python 3.9 oder neuer  (https://www.python.org/downloads/)
    → Bei der Installation "Add Python to PATH" aktivieren (Windows)
  - PySide6  (wird vom Installer automatisch installiert)

INSTALLATION — WINDOWS
  1. ZIP  maxscript-gui-editor-python-<version>.zip  entpacken.
  2. Doppelklick auf  install.bat
  3. Das Skript:
       • prüft die Python-Version
       • erstellt eine virtuelle Umgebung unter
           %LOCALAPPDATA%\MAXScriptGUIEditor\
       • installiert PySide6
       • erstellt Desktop-Shortcut und Startmenü-Eintrag
  4. Fertig — App über Desktop-Shortcut starten.

INSTALLATION — LINUX / MACOS
  1. ZIP entpacken.
  2. Terminal öffnen und ins entpackte Verzeichnis wechseln:
       bash install.sh
  3. Das Skript erstellt:
       • Virtuelle Umgebung unter  ~/.local/share/maxscript-gui-editor/
       • Starter-Skript unter      ~/.local/bin/maxscript-gui-editor
       • Linux: .desktop-Eintrag im Anwendungsmenü
       • macOS: App-Bundle unter   ~/Applications/
  4. App starten:
       maxscript-gui-editor        (Terminal)
       oder über das Anwendungsmenü

MANUELL STARTEN (ohne Installer)
  pip install PySide6
  python main.py

BEDIENUNG
  Linkes Panel    – Steuerelemente per Drag & Drop auf den Canvas ziehen
                    oder "Add →"-Button klicken
  Canvas (Mitte)  – WYSIWYG-Vorschau; Controls verschieben durch Ziehen
                    (Raster-Snap 4 px)
  Rechtes Panel   – 4 Tabs:
                      Control     – Name, Label, Position, Größe, Typ-Eigenschaften
                      Events      – Event-Handler mit Code-Editor
                      Rollout     – Rollout-Name, Titel, Breite
                      MacroScript – Ausgabemodus, Kategorie, interner Name
  Code-Bereich    – Unten: generierten Code anzeigen, kopieren, direkt zu Max senden

  Tastenkürzel:
    Ctrl+Z / Ctrl+Y   Undo / Redo (50 Stufen)
    F5                Code generieren
    F6                Code an 3ds Max senden (TCP-Bridge)
    Ctrl+Shift+C      Code in Zwischenablage
    Ctrl+D            Control duplizieren
    Del               Ausgewähltes Control löschen

SPEICHERN / LADEN
  Datei > Save      → Layout als .json speichern
  Datei > Open      → gespeichertes Layout laden
  Format: JSON (lesbar, versionierbar mit Git)

DEINSTALLATION — WINDOWS
  - Ordner löschen:  %LOCALAPPDATA%\MAXScriptGUIEditor\
  - Desktop-Shortcut manuell löschen.

DEINSTALLATION — LINUX / MACOS
  rm -rf ~/.local/share/maxscript-gui-editor/
  rm -f  ~/.local/bin/maxscript-gui-editor
  rm -f  ~/.local/share/applications/maxscript-gui-editor.desktop   (Linux)
  rm -rf ~/Applications/MAXScript\ GUI\ Editor.app                  (macOS)

==============================================================
  C) TCP-BRIDGE — LIVE-VERBINDUNG ZU 3DS MAX
==============================================================

Die Python-App kann generierten Code direkt in eine laufende
3ds Max-Instanz senden — kein manuelles Kopieren nötig.

BRIDGE STARTEN (in 3ds Max)
  Scripting > Run Script > max_bridge_listener.ms wählen.
  Der MAXScript Listener zeigt:
    === MAXScript Bridge Listener started on port 27120 ===

CODE SENDEN (in der Python-App)
  1. Code > Bridge Settings… — Host und Port prüfen
       Standard: 127.0.0.1 : 27120
  2. "⬤ Ping" — Verbindung testen (wird grün bei Erfolg)
  3. "▶ Send to Max" oder F6 — Code wird in Max ausgeführt

BRIDGE BEENDEN (in 3ds Max)
  Im MAXScript Listener eingeben:
    stopBridgeListener()

STANDALONE EXE ERSTELLEN (kein Python beim Endnutzer nötig)
  pip install pyinstaller
  pyinstaller maxscript_gui_editor.spec
  → Ausgabe: dist\MAXScriptGUIEditor\MAXScriptGUIEditor.exe
