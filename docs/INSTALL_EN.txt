MAXScript GUI Editor
====================
Visual Rollout & MacroScript Designer for Autodesk 3ds Max
----------------------------------------------------------
Two editions:
  A) MAXScript Edition  — runs directly inside 3ds Max
  B) Python Standalone  — desktop app with WYSIWYG canvas & TCP bridge

==============================================================
  A) MAXSCRIPT EDITION
==============================================================

REQUIREMENTS
  - Autodesk 3ds Max 2022 or newer

INSTALLATION
  1. Extract  maxscript-gui-editor-<version>.zip.
  2. Open 3ds Max.
  3. Run the installer — choose one method:
       • Drag  installer-<version>.ms  onto the viewport
       • Scripting > Run Script > select  installer-<version>.ms
  4. Read the confirmation dialog and close it.
     The tool is now registered as a MacroScript (category "Tools").

ADDING A TOOLBAR BUTTON
  - Customize > Customize User Interface > Toolbars tab
  - Category: Tools
  - Drag "MAXScript GUI Editor" onto any toolbar.
  Tip: Use the Keyboard tab to assign a keyboard shortcut.

USAGE
  Left panel   – "Add Control" buttons: choose a control type to add.
  Right panel  – Edit properties of the selected control.
  Bottom area  – Rollout settings, undo/redo, save, code generation.

  Add a control       → click a button in the left panel
  Select a control    → click it in the left list
  Edit properties     → right panel (type-specific)
  Event handlers      → "Event Handlers" tab in the left panel
  Generate code       → "Generate Code" → code copied to clipboard
  MacroScript mode    → enable in "MacroScript Config" tab

SAVE / LOAD
  "Save Layout"   → save design as a .dat file
  "Load Layout"   → load a saved design
  "Import Code"   → parse existing MAXScript rollout code
                    (basic parser — complex expressions may need manual adjustment)

UNINSTALL
  1. Delete the folder:
       %LOCALAPPDATA%\Autodesk\3dsMax\<version>\ENU\scripts\MAXScriptGUIEditor\
  2. Remove the toolbar button:
       Customize > Customize User Interface > right-click the button > Delete.

==============================================================
  B) PYTHON STANDALONE EDITION
==============================================================

REQUIREMENTS
  - Python 3.9 or newer  (https://www.python.org/downloads/)
    → On Windows: check "Add Python to PATH" during installation
  - PySide6  (installed automatically by the installer)

INSTALLATION — WINDOWS
  1. Extract  maxscript-gui-editor-python-<version>.zip.
  2. Double-click  install.bat.
  3. The script will:
       • check the Python version
       • create a virtual environment at
           %LOCALAPPDATA%\MAXScriptGUIEditor\
       • install PySide6
       • create a Desktop shortcut and Start Menu entry
  4. Done — launch the app from the Desktop shortcut.

INSTALLATION — LINUX / MACOS
  1. Extract the ZIP.
  2. Open a terminal in the extracted folder:
       bash install.sh
  3. The script creates:
       • virtual environment at  ~/.local/share/maxscript-gui-editor/
       • launcher script at      ~/.local/bin/maxscript-gui-editor
       • Linux: .desktop entry in the application menu
       • macOS: app bundle at    ~/Applications/
  4. Launch:
       maxscript-gui-editor        (terminal)
       or via the application menu

MANUAL START (without installer)
  pip install PySide6
  python main.py

USAGE
  Left panel      – drag controls onto the canvas from the palette
                    or click "Add →"
  Canvas (centre) – WYSIWYG preview; drag controls to reposition
                    (4 px grid snap)
  Right panel     – 4 tabs:
                      Control     – name, label, position, size, type properties
                      Events      – event handlers with code editor
                      Rollout     – rollout name, title, width
                      MacroScript – output mode, category, internal name
  Code area       – bottom: view, copy, or send code directly to 3ds Max

  Keyboard shortcuts:
    Ctrl+Z / Ctrl+Y   Undo / Redo (50 levels)
    F5                Generate code
    F6                Send code to 3ds Max (TCP bridge)
    Ctrl+Shift+C      Copy code to clipboard
    Ctrl+D            Duplicate selected control
    Del               Delete selected control

SAVE / LOAD
  File > Save   → save layout as .json
  File > Open   → load a saved layout
  Format: JSON (human-readable, Git-friendly)

UNINSTALL — WINDOWS
  - Delete:  %LOCALAPPDATA%\MAXScriptGUIEditor\
  - Delete the Desktop shortcut manually.

UNINSTALL — LINUX / MACOS
  rm -rf ~/.local/share/maxscript-gui-editor/
  rm -f  ~/.local/bin/maxscript-gui-editor
  rm -f  ~/.local/share/applications/maxscript-gui-editor.desktop   (Linux)
  rm -rf ~/Applications/MAXScript\ GUI\ Editor.app                  (macOS)

==============================================================
  C) TCP BRIDGE — LIVE CONNECTION TO 3DS MAX
==============================================================

The Python app can send generated code directly into a running
3ds Max session — no copy-paste needed.

START THE BRIDGE (in 3ds Max)
  Scripting > Run Script > select  max_bridge_listener.ms
  The MAXScript Listener prints:
    === MAXScript Bridge Listener started on port 27120 ===

SEND CODE (in the Python app)
  1. Code > Bridge Settings… — verify host and port
       Default: 127.0.0.1 : 27120
  2. "⬤ Ping" — test the connection (turns green on success)
  3. "▶ Send to Max" or F6 — code is executed in 3ds Max

STOP THE BRIDGE (in 3ds Max)
  Type in the MAXScript Listener:
    stopBridgeListener()

BUILD A STANDALONE EXE (no Python required at the target machine)
  pip install pyinstaller
  pyinstaller maxscript_gui_editor.spec
  → Output: dist\MAXScriptGUIEditor\MAXScriptGUIEditor.exe
